### Where the extractors didn't help

- **C-category UI fixes (P9, P10, P11)** — the fe-component nodes exist in the
  graph but the queries are about *styling* ("3 точки", "right align",
  "drawer right side"), which embedding similarity cannot bridge to component
  names. Baseline doc called this out explicitly as out-of-scope.
- **P14 multi-tenancy** — cross-cutting concept spread across 50+ files;
  scoped-marker nodes alone are too few to anchor it. Baseline doc warned this
  would not be addressed.
- **B1, B4 (checkout, payments)** — these features simply don't exist in
  project-c; no extractor can fabricate nodes.
- **P4 ("карты")** — multilingual embedding collapsed "карты" (maps) and
  "cards" (UI cards), and there is no ReminderMap component to anchor to.
