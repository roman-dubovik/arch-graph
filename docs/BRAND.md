# arch-graph — mini brand kit

**One-liner:** Source code in, typed graph out — deterministic, sub-second.

## Voice
Engineering-precise. Static, deterministic, structural, layered, topological. No AI-magic, no playful mascot, no organic curves.

## Logo concept
A stylized **A** built from graph primitives: three angular nodes, two diagonal edge legs, one horizontal edge/crossbar, and two junction nodes. From distance: `A`. Up close: typed mini-graph.

## Colors
- Background: `#0A0A0F`
- Primary cyan: `#7DD3FC`
- Indigo accent: `#8B9EFF`
- Text: `#F8FAFC`
- Muted text: `#94A3B8`

## Usage sizes
- Favicon: 16×16 / 32×32
- Header: 22×22
- Flow animation center: 80×80
- Hero / social: 200×200+

## Rules
- Prefer mono-color stroke for product UI.
- Use 2-tone only in hero/social contexts.
- Keep sharp caps and miter joins.
- Avoid generic graph blobs, gradients, organic splines, sparkles, AI-style glows.
